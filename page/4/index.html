財務報表分析 - 第四章 財務報表分析的技術、常有的財務比率

2021-10-24
北科上課筆記 / 財務報表分析

將資產負債表及綜合損益表轉換成共同比財務報表

  • 資產負債表: 以資產總額為 100%，各項金額除以資產總額就是各項比率
  • 綜合損益表: 以營業收入(銷貨淨額)為 100%，各項金額除以銷貨淨額就是各項比率
    • 台積電 2019 年簡明損益比較分析

舉例：台積電的獲利之趨勢分析


  • 以 2013 為 base，去對 2014~2019 做比例
  • 營業收入淨額、營業毛利、營業利益、本期淨利都是往上，表示這公司獲利良好，是一間好公司。
  • 但近期向上幅度變小，可能有點沒力，需要密切注意下一年的狀態。

常用的財務比率

變現力比率

  • 流動比率 \(\frac{流動資產}{流動負債}\)
  • 速動比率 \(\frac{速動資產}{流動負債} = \frac{流動資產 - 存貨 - 預付費用}{流動負債}\)

負債管理比率

  • 負債比例 \(\frac{負債總額}{資產總額}\)
  • 負債對股東權益比率 \(\frac{負債總額}{股東權益總額}\)
  • 權益乘數 \(\frac{資產總額}{股東權益總額}\)
  • 利息保障倍數 \(\frac{稅前息前淨利}{利息費用}\)
  • 現金涵蓋比率 \(\frac{稅前息前淨利+折舊費用}{利息費用}\)

資產管理比例

  • 存貨周轉率 \(\frac{營業成本}{平均存貨}\)
  • 存貨週轉天數 \(\frac{365天}{存貨周轉率}\)
  • 應收帳款周轉率 \(\frac{賒帳收入淨額}{平均應收帳款}\)
  • 平均收現期間 \(\frac{365天}{應收帳款週轉率} \)
  • 不動產、廠房及設備周轉率 \(\frac{營業收入淨額}{平均淨不動產、廠房及設備}\)
  • 總資產周轉率 \(\frac{營業收入淨額}{平均總資產}\)

獲利能力比率

  • 純益率 \(\frac{稅後淨利}{營業收入淨額}\)
  • 股東權益報酬率 \(\frac{稅後淨利}{平均股東權益}\)
  • 總資產報酬率 \(\frac{稅後淨利 + 利息費用 * (1- 邊際稅率)}{平均總資產}\)

市場價值比率

  • 本益比 \(\frac{普通股每股市價}{每股盈餘}\)
  • 股利收益率 \(\frac{每股股利}{普通股每股市價}\)
  • 市價淨值比 \(\frac{每股股價}{每股淨值}\)

演算法知識 - Maximum Bipartite Matching (使用 Augmenting Path Algorithm)

2021-10-16
演算法知識

Maximum Bipartite Matching 介紹

在介紹最大二分匹配時，必須先介紹二分圖。
二分圖是一種圖的特例，二分圖的結構為，X 群體的每一個點都有連結到至少一個以上 Y 群體的點、T 群體的每一個點都有連結到至少一個以上 X 群體的點，且 X、Y 群體各自沒有邊互相連接。

圖源引用師大演算法

二分匹配就是，每一個 x 節點只能連到一個 y 個節點、每一個 y 節點只能連到一個 x 個節點，舊式二分匹配，類似於一夫一妻制。

圖源引用師大演算法

我們使用 Augmenting Path Algorithm 實作，時間複雜度為 \(O(VE)\)，V 是頂點、e 是邊
