演算法知識 - Segment Tree 線段樹

2021-03-06
演算法知識

Segment Tree 線段樹 介紹

線段樹是一種二元樹型態的資料結構，通常用於需要大量查詢區間的問題，空間複雜度是 \(O(n)\)，查詢的時間複雜度則是 \(O(log \ n + k\)，k 為符合條件的區間數量

這種資料結構也能夠擴展到高維度
