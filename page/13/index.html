UVa

題目大意：

type 2 ，移除字串從 p 位置開始移除 c 個字元
type 3 ，查詢 v 版本從 p 位置開始輸出 c 個字元

我們順便進行加密混淆的動作，每一次只要 type 3 輸出的字元中擁有 ‘c’，則接下來 除了 type 以外的數字都會因為前面 type 3 輸出字元中有多少的 c，則會增加多少數字。
