UVa526 - String Distance and Transform Process (Minimum Edit Distance 詳解)

2020-10-04
UVa / 演算法知識

題目大意：

UVa11475 - Extend to Palindrome (KMP algorithm 詳解)

2020-10-03
UVa / 演算法知識

題目大意：

給你一字串，請新增字元讓這字串變成迴文，但新增字元數量要最少。
迴文：從左邊讀與從右邊讀意思都一樣
題目善意提示：這題不要給經驗不足的新手做 ^^
