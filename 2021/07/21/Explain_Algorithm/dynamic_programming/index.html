動態規劃 Dynamic programming 介紹

動態規劃用在需要狀態轉移的題目上，例如說 \(y = 2x+1\)，那麼在計算出 y 之前就必須要有 x，否則 y 沒有辦法產生一個自然數。

動態規劃最好的理解方式就是，我們將某些變數的所有可能都列舉出來，並根據我們之前的紀錄，並透過遞迴關係式找出答案。
舉例：\(y = 2x+1\)，我們可以將產生出來的 y，再帶入 x，想請問 y 等於 15 有幾種組合可以產生。
