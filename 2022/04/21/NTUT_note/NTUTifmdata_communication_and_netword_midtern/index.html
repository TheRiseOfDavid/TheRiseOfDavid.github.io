筆記說明

此筆記用途在於台北科技大學資訊與財金管理系大三下企業資料通訊重點整理
並非所有人都適用，部分對我而言稍加容易的內容並不會寫在此內。

CH1 介紹

Q1-1 為何需要協定？

協定定義了如何溝通，何時溝通，準確的傳送資料給不同的 device

Q1-3 說出 4 種基本網路拓撲，並指出其優點。

a. Mesh 安全
b. Bus: 安裝簡單
c. Star: 強大網路
d. Ring: 易於故障隔離

Q1-5 當有住戶使用撥接或 DSL 服務連上 Internet，電話公司的角色為何？

電話公司腳色為 ISP，從住戶到電話公司使用 WAN 連線到 Interent.

Q1-7 多點連線優於點對點連線的地方為何？

易於安裝且低成本

Q1-9 指出資料通訊系統的 5 個元件。

sender, recevier, transmission medium, message, protocol

Q1-11 以線路組態的方來歸類 4 個基本拓撲。

point-to-point: mesh ring star
multiple point: bus

Q1-13 如果每一條 LAN 都必須直接與另一條 LAN 通訊，那麼需多少條點對點 WAN 來連接 n條 LAN？

\( n * (n-1) / 2)\)

Q1-15 解釋 Internet 草案和建議標準的差異。

Interent draft 的文件非正式且只有六個月的生命週期，
Interent draft 中受到大量關注並提案，接受後就變成 Internet 標準

Q1-17 決定通訊系統是 LAN 或 WAN 的因素為何？

大小、距離、架構、與所有權

Q1-19 說明 IETF 和 IRTF 的職責差異

IETF identify operatioal problem and recommend solution
IRTF focuse long-terms topic.

CH2 網路模型

Q2-1 下列各層使用的位址型式為何？ a. 應用層 b. 網路層 c. 資料鏈結層

a. 通常是用來找到目的低的電腦名稱 EX: something@somewhere.com
b. 使用 two logical address source and destination 去定義 source and 目標電腦，address 為獨特的
c. 使用 two link address source and destination 去 link 起點到終點

Q2-3 你能解釋為何應用層中沒有提到多工 / 解多工的服務？

由於應用層上面並沒有其他 layer，因此不需要提供服務給其他人，所以就沒有多工需要

Q2-5 本章討論協定分層必須遵循以使通訊能夠雙向的第一個準則為何？

每一個 layer 都必須提供兩個兩個 task，each diresct using one task.

Q2-7 下列資料單元中哪一個是使用者資料元的解封裝？ a. 資料元 b. 區段 c. 訊息

transpot-layer 解封裝訊息給應用層

Q2-9 有一主機以 TCP/IP 協定組與另一台主機通訊，下列各層所收送的資料單元為何？ a. 應用層 b. 網路層 c. 資料鏈結層

a. message
b. datagram
c. frame

Q2-11 列出一些本章提及的應用層協定。

HTTP、FTP、SSH、DNS

Q2-13 路由器連接 3 條鏈結（網路），路由器會牽涉到下列哪些層次？ a. 實體層 b. 資料鏈結層 c. 網路層

3 個實體層
3 個資料連結層
1 個網路層

Q2-15 如果兩端主機間只有一條路徑，兩主機間需要一台路由器嗎？

不需要，因為只有一個 path，路由器是在找最佳路徑時才要用到

CH3 實體層介紹

Q3-1 複合訊號如何分解成各單獨的頻率？

Fourier series 在一個週期訊號內給一個頻域
Fourier analysis 在一個週期訊號內給定一個頻域

Q3-3 週期與頻率之間的關係為何？

互為倒數

Q3-5 為何光纖中的光訊號其波長很短？

因為光的頻率極快，因此波長才這麼低

Q3-7 我們從麥克風送出聲音到錄音機，這是基頻傳輸或寬頻傳輸？

baseband 因為我們沒有做 modulation

Q3-9 奈奎斯特定理與通訊有何關聯？

The Nyquist theorem 定義最大的 bit rate 在無 noise channel

Q3-11 語音訊號的頻域圖是離散或連續？

連續的，因為聲音是非週期性訊號

Q3-13 如何分辨基頻傳輸和寬頻傳輸？

Baseband transmission 沒有做 modulation using low-pass channel.
Broadband transmission 則是有 modulation using band-pass channel

Q3-15 我們調變許多語音訊號，並送到空氣中，這是基頻傳輸或寬頻傳輸？

寬頻傳輸，因為有被空氣 modulation.

CH4 數位傳輸

Q4-1 列出本書討論過的五種線路編碼機制。

unipolar, polar, bipolar, multilevel, multi-transition

Q4-3 定義攪亂碼並舉出它的目的

Scrambling, 替代 zero-level pulses 並 conbination of other levels.

Q4-5 分辨資料速率和訊號速率

data rate: 多少個 data element sent in 1s.
bits per second bps: 多少個 signal elements in 1s. 單位是 band

Q4-7 平行傳輸和序列傳輸之間的差異為何？

In parallel transmission: 可以在 1s 中送好幾個 bits
serial transmission: 一秒一個

Q4-9 列出三種數位對數位轉換的技術。

line coding, block colding, scrambling

Q4-11 定義 DC 成分和它對數位傳輸的效應。

當電壓訊號保持一定狀態時，頻譜就會產生非常低的頻率，有些系統沒辦法通過低頻時就會出現問題

CH5 類比傳輸

Q5-1 定義數位對類比轉換

一種程序，將數位訊號資料改變為類比資料；代表基頻數位訊號調幅成寬頻類比訊號

Q5-3 下列類比對類比轉換中，類比訊號的哪些特性改變以表示低通類比訊號？ a. AM b.FM c. PM

  • AM change the amplitude(震幅)
  • FM change the frequency(頻率)
  • PM change the phase(階段)

Q5-5 定義類比訊號

類比傳輸使用 band-pass channel
baseband(基頻) 中的類比、數位訊號轉換成複雜的類比訊號頻率，並且在適合的頻道中

Q5-7 星宿圖上的訊號，其兩個分量為何？哪一個分量代表水平軸，哪一個分量代表垂直軸？

兩個 components I、Q
I 表示水平軸、Q 表示垂直軸

Q5-9 四種數位對類比轉換 (ASK, FSK, PSK, QAM) 的技術中，哪一個最容易受雜訊的影響？請說明。

ASM 最容易受到 noise 影響
