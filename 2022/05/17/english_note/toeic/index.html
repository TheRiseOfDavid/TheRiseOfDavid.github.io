說明

記錄大衛在 TOEIC 學習中，所注意的事項，並將它做成筆記

Part1

  • 注意時態
    • 現在簡單式，表達事實、狀態
    • 現在進行式，表達當下正在發生的動作
  • 聽力，大家低頭開始寫的選項，你也跟著寫

Part2

  • 四大重點
    • 時態
    • 相似發音
    • 一樣的單字
    • 一字多義
  • 句子特性
    • 否定句
      • not A or B
      • neither A nor B
    • 肯定句
      • both A and B
      • either A or B
      • not only A but also B

Part3

  • 開始念第三大題的指示時，別浪費時間要先看第一部分的題目+選項。
  • 聽力開始唸該大題的題目時，要趕快看下一大題的題目+選項，依此類推。
  • 看題目要注意
    • who, when (what time, what day, what date…), where (which place), how long, how much…這些疑問詞開頭的問句，因為問題明確所以疑問詞要先圈裡來，動作的部分畫底線。
    • 若疑問詞不明確時(what, why, how)，記得把動作的部分畫底線
  • 基本上，題目的順序會按照聽力播放的順序來出題，若沒聽到任何一題，就跳過隨便猜一個最有可能(跟題目不甚相關)的答案，然後趕快跟上聽力播放的腳步答題。

Part4

none

Part5

形容詞 adj, 副詞 adv

  • a(n)/this/that/the/所有格/no + (adj) + N
  • 形容詞 adj + N slow move / hard work
  • V + 副詞 adv move slowly / work hard
  • 程度副詞 adv + 形容詞 adj ex: very/really/too/so/quite +cute

動詞 V

  • 助動詞 + 原V
  • 不定詞 to + 原 V
    • *當 to 為介係詞時 + Ving *
    • want to + 原V
    • would like to + 原V
    • would love to + 原V
    • feel like + Ving

N

1 2 3 4 5
主格 I we you he/she/it they
所有格 my our your his/her/its their
受格 me us you him/her/it them
所有格代名詞 = 所有格+名詞 mine ours yours hims/hers/its theirs

Part6

  • 當動詞為來去動詞(come, go, arrive, leave)或是表預定的計畫(plan, visit…)時，可以使用現在進行式代替未來式。
  • 時態表格解釋
    11/23 12/23 01/23 02/23
    0 1/3 2/3 3/3
    現在式 have/has been Ving have/has been Ving have/has PP

-

  • 副詞擺放位置
    • beV之後
    • 一般動詞之前
    • 夾中間:
      • beV + 情狀副詞 + Ving
      • 助動詞 + 情狀副詞 + 原V
      • have/has + 情狀副詞 + p.p.
  • 規則的情狀副詞(adj~ly): 句首 or 句尾(動作結構之後)
  • adj / adv : 原級/比較級/最高級
  • 原級: A + beV/連綴V + as adj原級 as + B.
  • 比較級: A + beV/連綴V + adj比較級(單音節: adj er / 多音節: more + adj原級) + than + B.
  • much/a lot/even/still/a little/quite/far + adj比較級(單音節: adj er / 多音節: more + adj原級)
    • O: much more interesting
    • X: more much interesting
    • X: more taller
    • O: much taller
    • X: more much taller
  • 最高級: A + beV/連綴V + the + adj最高級(單音節: adj est / 多音節: most + adj原級) + 範圍.

Part7

Reading:

  • 先看題目:
    • skimming: who, where, when(what time, what day, what date)
    • scanning: why, what, how
  • 注意題目的順序
  • 看清楚標題 (電子郵件；公告 subject)
  • 注意文章內小字 (公告；表格或有星號註記的部分)
  • letters: 英文書信都用破題法，信件的重點(目的)都在第一段
