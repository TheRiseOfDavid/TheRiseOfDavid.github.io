筆記說明

此筆記用途在於台北科技大學資訊與財金管理系大一下財務管理中債劵定價重點整理
並非所有人都適用，部分對我而言稍加容易的內容並不會寫在此內。

名詞解釋
  • r = 利率
  • t = 期數

債劵定價

  • 殖利率 (YTM)
    • 每年使用這筆資金平均每年應該要付給債權人的資金成本
    • 會隨著其他因素改變，如:違約程度風險，債劵到期時間長短
    • \(公司債券 = \sum_{t=0}^{n} \frac{利息}{(1+殖利率)^{t}} + \frac{本金}{(1+殖利率)^{n}} \)
    • \(YTM = 當期受益率 + 資本利得(損失)率 \)

殖利率計算機按法

  1. set = end
  2. PV 輸入發行價格並在前面加上負號，如: -123
  3. PMT 輸入利息
  4. FV 輸入票價
  5. 到 i 點 solve
  • 票價發行
    • 票面利率與殖利率比較 評價
      票面利率 = 殖利率 評價發行
      票面利率 > 殖利率 溢價發行
      票面利率 < 殖利率 折價發行
  • 債劵風險
    • 到期日越久的債劵風險越高
    • 利率越高的債券風險越低
  • 當期收益率 \( current yield \)
    • \(current yield = \frac{當年度利息收入}{債劵當年度期初價格} \)
  • 資本利得率
    • \(資本利得率 = \frac{債券今年價格 - 債券去年價格}{債券去年價格} \)

債券特性

  • 債券的利息可以被當成費用，降低稅率
  • 債券名稱依時間長短區分
    • 中期債券 -> 一年以上，十年以下
    • 長期債券 -> 十年以上
  • 債劵名稱以擔保品區分
    • 擔保債券 -> 用股票或其他有價債券擔保
    • 抵押擔保債券 -> 以固定資產或不動產擔保
    • 信用債券 -> 用信用擔保，通常信譽良好大公司才能發行
      • 信用債券必須等到公司將其他債務清償完畢後才能求償，但能優於股東
  • 贖回債券 (call provision)
    • 公司可以買回，但通常會比票面價格高
  • 償債基金 (sinking fund)
    • 透過特定信託公司，買回或贖回特定債券

債券評價

  • 債券平等
    • 評價越高，投資人要求報酬低
    • 評價越低，投資人要求報酬高
  • 投資型債券
    • 評價等級高，以賺取本金與少許利息
  • 投機型債券
    • 評價等級低，以賺取本金與大量利息

市場術語

  • 交叉型債券
    • 被 n 種信評公司評初不同等級債券
  • 垃圾債券 (Jund Bond)
    • 信用較差公司，發行的高收益債券，通常都以高票面利率為主
  • 墮落天使 (fallen angel)
    • 從投資型債券轉變成垃圾債券
  • NCAA
    • No coupon at all -> 無法支付任何利息的債券

其他債券

  • 金融債券
    • 還本期間不得低於兩年
  • 零息債券 (zero coupon bond)
    • 完全不支付利息，到期末時將一次性將面額償還，通常面額裡面內含利息
    • \(發行價格 = \frac{發行面額}{(1+r)^{t}}\)
    • 發行必定為折價發行
    • \(利息(coupon) = 發行面額 - 發行價格 \)
    • 到期前不付息，但必須提列利息費用
  • 浮動利率 (floating-rate bond)
    • 每 3 到 6 個月進行調整利率
    • 仍有上下限，上下限區間為利率區間
    • 通常為金融機構發行，時間約為 10 年
  • 可轉換債券 (convertible bond)
    • 在到期前，可以有權利(非義務)轉成原發行機構股票
  • 可交換公司債 (exchangeable bond)
    • 在到期前，可以有權利(非義務)轉成其他發行機構股票
  • 附認購權債證券 (warrant bond)
    • 在購買此債券時，可以用較低價格買該機構發行股票，以刺激投資債券
