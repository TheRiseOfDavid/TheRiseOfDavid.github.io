DataSet 名詞介紹

  • Elements 單位名稱，如:公司名稱
  • Variables 想查看的變數，如:公司盈餘
    • 變數歸類，如：「種類」、「數字」、「有無」
  • Observations 單筆元素的變數資料
  • DataSet 全部元素的變數資料，不包含 Elements and Variables

Scales of Measurement 衡量此度 *必考

  • Nominal 名目尺度，歸類用。
    只能計算次數，不能做其他使用。
  • Ordinal 順序尺度，歸類用
    只能知道相對程度。
  • Interval 等距尺度，可進行分析
    加減可以進行分析。
  • Ratio 等比尺度，可進行分析
    加減乘除都可以進行分析。
