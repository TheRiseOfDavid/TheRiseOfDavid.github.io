Discrete Random Variable 離散隨機變數

  • Finite Number of Values
    在已知的集合中隨機抽出的元素。
  • Infinite Sequence of Values
    在無限的集合中隨機抽出的元素，此集合大小為Infinite。

Discrete Probability Distributions 離散機率

  • 用指派來指派每一個元素的機率
  • 用公式推出，且要符合此公式 \(f(x) \geq 0\) and \( \sum_{}^{} f(x) = 1 \)
